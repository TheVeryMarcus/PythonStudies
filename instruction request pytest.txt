 инструкция для request в PyCharm :

### Шаги для выполнения кода в PyCharm:

1. **Создайте новый проект или откройте существующий**:
   - Если у вас еще нет проекта, создайте новый, выбрав "Create New Project" в стартовом окне PyCharm.
   - Если проект уже есть, откройте его через "Open" в стартовом окне.

2. **Создайте новый Python-файл**:
   - В панели Project в правой части окна PyCharm щелкните правой кнопкой мыши на папке вашего проекта.
   - Выберите `New` -> `Python File`.
   - Назовите файл, например, `test_requests.py`.

3. **Вставьте код в файл**:
   - Скопируйте и вставьте следующий код в созданный файл:
  
     import requests

     response = requests.get('https://api.github.com')
     print(response.status_code)
     

4. **Убедитесь, что библиотека `requests` установлена**:
   - Если библиотека `requests` еще не установлена, вы можете установить её через терминал PyCharm.
   - Откройте терминал в PyCharm (обычно находится в нижней части окна).
   - Введите команду:
      
     pip install requests
 

5. **Запустите скрипт**:
   - Щелкните правой кнопкой мыши на вкладке с вашим файлом `test_requests.py`.
   - Выберите `Run 'test_requests'` или используйте сочетание клавиш для запуска (обычно `Shift + F10`).

6. **Проверьте результат**:
   - В нижней части окна PyCharm откроется вкладка "Run", где вы увидите результат выполнения скрипта. Вы должны увидеть `200`, что означает успешный запрос к API GitHub.



 инструкция для pytest в PyCharm :

Чтобы установить библиотеку `pytest` в вашей среде разработки, выполните следующие шаги:

### Установка `pytest`

1. **Откройте терминал или командную строку**:
   - В PyCharm вы можете использовать встроенный терминал, который обычно находится в нижней части окна.


2. **Установите `pytest`**:
   - Введите следующую команду и нажмите Enter:
  
     pip install pytest
 

3. **Проверка установки**:
   - После установки вы можете проверить, что `pytest` установлен правильно, создав небольшой тестовый файл.
   - Создайте файл с именем, например, `test_example.py`, и добавьте в него следующий код:
 
     def test_example():
         assert 1 + 1 == 2
     
   - Запустите тесты с помощью команды:
      
     pytest test_example.py
     
  - Если всё настроено правильно, вы увидите сообщение о том, что тест прошел успешно.

### Установка в виртуальном окружении

Если вы используете виртуальное окружение, убедитесь, что оно активировано перед установкой `pytest`.

- **Активация виртуального окружения**:
  - На Windows:
    
    .\venv\Scripts\activate
   
  - На macOS и Linux:
    
    source venv/bin/activate


- **Установка `pytest` в виртуальном окружении**:
  
  pip install pytest
